# TCCAA Website

This is a Jekyll-based website for the **Tri-Cities Chinese American Association (TCCAA)**, hosted at [www.tccaa.org](http://www.tccaa.org).

## About TCCAA

The Tri-Cities Chinese American Association was established in 1975 to support the growing Chinese community in the Tri-Cities area of Washington State. The organization's mission is to:

- Enhance communication and foster friendships among members
- Promote cultural exchange between Chinese and American communities
- Help the local community understand Chinese culture and society
- Support community growth and family life

## Website Structure

This is a standard Jekyll site using the Minimal Mistakes theme with the following structure:

### Key Files
- `_config.yml` - Main Jekyll configuration
- `index.markdown` - Homepage with pinned posts, recent posts, and event calendar
- `about.markdown` - About page with TCCAA history and mission
- `CNAME` - Custom domain configuration for www.tccaa.org

### Content Organization
- `_posts/` - Blog posts about events and announcements (bilingual English/Chinese)
- `_layouts/` - Page templates (splash, post, default)
- `_includes/` - Reusable HTML components
- `assets/` - Images, pages, and other static content
- `_data/navigation.yml` - Site navigation configuration

### Features
- **Bilingual Content**: Posts and pages in both English and Chinese
- **Event Calendar**: Embedded Google Calendar showing community events
- **Responsive Design**: Uses Minimal Mistakes theme for mobile-friendly layout
- **Community Pages**: Donation acknowledgments, bylaws, committee information

## Content Types

The site regularly publishes posts about:
- Community events (Lunar New Year celebrations, Dragon Boat Festival, etc.)
- Membership information and committee updates
- Cultural celebrations and community activities
- Announcements and important notices

## Technical Details

- **Platform**: Jekyll static site generator
- **Theme**: Minimal Mistakes (via remote_theme)
- **Hosting**: GitHub Pages
- **Domain**: Custom domain www.tccaa.org via CNAME
- **Default Layout**: Splash layout for most pages
- **Plugins**: jekyll-feed, jekyll-remote-theme

This website serves as the primary online presence for TCCAA, providing information about events, community history, and ways for members to stay connected.
